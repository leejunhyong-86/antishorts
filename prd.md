# AntiShorts - 제품 요구사항 문서

## 📋 개요
AntiShorts는 YouTube Shorts와 Instagram Reels 등의 숏폼 비디오를 다운로드하고 관리할 수 있는 데스크톱 애플리케이션입니다.

## 🎯 목표
사용자가 숏폼 콘텐츠를 쉽게 다운로드하고, 체계적으로 관리하며, 향후 편집 등의 작업을 수행할 수 있는 통합 플랫폼을 제공합니다.

## 👥 타겟 사용자
- 콘텐츠 크리에이터
- 소셜 미디어 마케터
- 비디오 아카이빙이 필요한 사용자
- 오프라인 시청을 원하는 사용자

## ✨ 1단계 핵심 기능

### 1. URL 기반 비디오 다운로드
**요구사항:**
- YouTube Shorts URL 지원
- Instagram Reels URL 지원
- URL 자동 감지 및 플랫폼 식별
- 다운로드 진행률 표시
- 에러 핸들링 및 사용자 피드백

**기술적 고려사항:**
- URL 파싱 및 검증
- 플랫폼별 다운로드 로직
- 네트워크 에러 처리
- 다운로드 재시도 메커니즘

### 2. MP4 파일 추출 및 저장
**요구사항:**
- 다운로드된 비디오를 MP4 형식으로 변환/저장
- 고품질 비디오 우선 다운로드
- 파일명 자동 생성 (중복 방지)
- 로컬 저장소 관리

**기술적 고려사항:**
- 비디오 포맷 변환 (필요시)
- 파일 시스템 구조
- 저장 경로 설정
- 디스크 공간 관리

### 3. 데이터베이스 저장 및 관리
**요구사항:**
- 비디오 메타데이터 저장
  - 제목
  - 설명
  - 업로더/채널명
  - 다운로드 날짜
  - 파일 경로
  - 플랫폼 (YouTube/Instagram)
  - 비디오 길이
  - 썸네일 URL
- 비디오 목록 조회
- 검색 및 필터링
- 비디오 삭제 (파일 + DB 레코드)

**기술적 고려사항:**
- 데이터베이스 스키마 설계
- 인덱싱 전략
- CRUD 작업
- 데이터 무결성

### 4. 사용자 인터페이스
**요구사항:**
- URL 입력 필드
- 다운로드 버튼
- 진행률 표시
- 비디오 라이브러리 그리드/리스트 뷰
- 비디오 썸네일 미리보기
- 비디오 재생 기능
- 검색 바
- 필터 옵션 (플랫폼별, 날짜별)

## 🔮 2단계 계획 (향후 기능)
- 비디오 편집 (자르기, 병합, 필터)
- 일괄 다운로드
- 재생목록 지원
- 태그 시스템
- 즐겨찾기
- 내보내기 기능
- 클라우드 동기화

## 🚫 제외 사항 (1단계)
- 비디오 편집 기능
- 클라우드 저장
- 사용자 계정 시스템
- 소셜 공유 기능

## 📊 성공 지표
- 다운로드 성공률 > 95%
- 평균 다운로드 시간 < 30초 (1분 비디오 기준)
- UI 응답 시간 < 100ms
- 데이터베이스 쿼리 시간 < 50ms

## 🔒 보안 및 법적 고려사항
- 저작권 준수 안내 메시지
- 개인 사용 목적 명시
- 다운로드 제한 (플랫폼 이용약관 준수)
